# 快速起步

## 第一步：建立帳號

打開瀏覽器，前往 `http://localhost:8000/auth/register`，填入 Email 與密碼（至少 8 字元），按「註冊」。註冊後自動登入，進入儀表板。

## 第二步：新增第一家公司

點上方導覽列的「**公司**」→「**+ 新增公司**」，填入名稱、產業、客戶等級後儲存。

客戶等級說明：

| 等級 | 意思 |
|------|------|
| 有規劃 | 客戶有明確採購計畫 |
| 有需求 | 有需求但尚未規劃 |
| 沒規劃 | 目前沒有需求 |
| 放棄 | 不再追蹤 |

## 第三步：補充公司的地址與電話

點公司名稱進入詳情，切到「**地址電話**」tab，分別新增總機號碼與辦公地址。

## 第四步：新增聯絡窗口

在詳情頁切到「**聯絡窗口**」tab，或從上方導覽列點「**聯絡人**」→「**+ 新增聯絡人**」，填入姓名、職稱、手機、Email，並選擇所屬公司後儲存。

> 聯絡人換公司時，直接編輯「所屬公司」欄位即可，歷史記錄不會消失。

## 第五步：記錄第一筆聯絡記錄

在公司詳情頁切到「**聯絡記錄**」tab，填入：

- **日期**：本次聯絡日期
- **內容**：對話摘要或備忘
- **狀態**：未接 / 接通沒談 / 寄送自介信…等
- **聯絡窗口**：勾選這次接觸的對象

按「新增記錄」儲存。重要記錄可以**置頂**，方便往後查閱。

## 第六步：貼興趣標籤與提醒日期

在公司詳情頁的「**基本資料**」tab 底部，找到標籤區塊，填入標籤名稱（例如「MDS」）與提醒日期（例如客戶表示可以聯絡的時間）。到期前，儀表板的「即將到期提醒」面板會顯示這筆提醒。

---

## 接下來

**批次匯入公司**：公司列表頁右上角點「**匯入 CSV**」，上傳 CSV 檔後預覽欄位對應，確認後一次匯入。支援欄位：公司名稱（必填）、產業、客戶等級。名稱重複的列會自動略過。

**匯出聯絡人到手機**：在聯絡人詳情頁點「**下載 vCard**」，取得 `.vcf` 檔，匯入手機通訊錄後即可直接撥號。

**查看全部聯絡記錄**：點導覽列「**聯絡記錄**」進入全域流水帳，可依狀態、公司、聯絡人、日期範圍篩選，快速找出近期應追蹤但還未聯絡的客戶。
